UHF RFID Readers

RDL UHF reader device is enabled with the world’s best high performance chip and Higher Sensitivity antenna to continuously monitor, track,

identify items and their locations within large areas.

RDL UHF Reader comes with features: fast identification, excellent multi-tag reading, strong anti-collision. IP65 rated UHF readers are widely used

in production automation, parking automation, fixed asset management, hospital, access control, logistics, warehouse inventory, product tracking

and other domains.

Features
  • Software SDK / API help in faster application development.
  • Support Ethernet, WiFi, 4G/LTE RS232, RS485, Wiegand
  • communication Interface*.
  • Supports TCP/IP, HTTP & MQTT protocol and faster data
  • processing*.

  • Support covering larger area over TCP/IP wired / wireless network
  • environment.

  • Aluminum casting / IP65 rated housing applies to all weather
  • industrial scenarios.
  • Deployment / wiring made easier with integrated Reader and antenna design.
  • Various SDK (.Net, Java, Python and Android) for rapid

    implementation of large projects.

  • Multiple reader types available, including handheld, wall- and desktop models.
  • Reading range supports 50 cm to 15 meters**.
  • Frequency 902-926 MHz , 865-868M Hz.
  • Read rate 200 count per second.
  • Adjustable Antenna (circular polarization) Range 3 DBi to 12 DBI and RF power up to 30 dBm*.
  • Supported ISO18000-6C(EPC Gen2).
  • Working Voltage range DC 9 to 24V.
